Chapter 10 — Free Types

10.1 The Natural Numbers

I. Concept

  • Free types offer concise, structured alternatives to verbose set/relation models.
  • Building nat: start with a zero element and a successor function succ.
  • First attempt: succ as a partial function → allows loops and dead-ends.
  • Exclude base from range: {zero} ∩ ran succ = ∅ → still allows dead-end elements with no successor.
  • Make succ a total function: succ : nat → nat → still allows collisions (two predecessors map to same successor).
  • Make succ injective: succ : nat ↣ nat → still allows extra “phantom” elements beyond the chain.
  • Smallest set requirement: nat must be the minimum set containing zero and closed under succ.

10.2 Free Type Definitions

I. Concept

  • Free type definition: special mechanism for introducing sets like nat.
  • Enumerated types: Colours ::= red | orange | yellow | green | blue | indigo | violet — seven distinct constants.
  • Abbreviation vs. free type: Colours == {red, …} does not introduce names or guarantee distinctness; ::= does both.
  • Constructor functions: FreeType ::= constructor «source» — one element per source value, constructor is injective.
  • Recursive types: source may refer to the type itself → nat ::= zero | succ «nat».
  • Inference rule 1 — disjointness: all constants are distinct; constructor ranges are pairwise disjoint.
  • Inference rule 2 — closure/induction: any subset containing all constants and closed under all constructors must equal the entire type.
  • Order independence: the order of alternatives in a ::= definition does not matter.

II. Vocab Table

10.3 Proof by Induction

I. Concept

  • Induction principle: derived from the closure inference rule of a free type.
  • Characteristic set: given predicate P on type T, define S == {t : T | P t}.
  • From closure: if S contains all constants and is closed under constructors → S = T → ∀ t : T • P t.
  • General template: prove P c1, …, P cm (base cases) and ∀ e • P(di e) (inductive steps) → conclude ∀ t : T • P t.
  • nat induction: P zero and ∀ m : nat • P m ⇒ P(succ m) → ∀ n : nat • P n.
  • Tree induction: ∀ n : ℕ • P(leaf n) and ∀ l, r : Tree • P l ∧ P r ⇒ P(branch(l, r)) → ∀ t : Tree • P t.
