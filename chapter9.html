Chapter 9 — Sequences

9.1 Sequence notation

I. Concept

  • A sequence is an ordered collection — written ⟨ a, b, c ⟩.
  • Empty sequence: ⟨⟩ — no objects.
  • Concatenation: s ⁀ t — elements of s followed by elements of t.
  • Filter: s ↾ A — keep only elements in set A, preserve order.
  • Head / tail: head s = first element; tail s = remainder.
  • Length: #s — number of elements; #(s ⁀ t) = #s + #t.
  • Distributed concatenation: ⁀/ flattens a sequence of sequences.
  • Reconstruction: s = ⟨ head s ⟩ ⁀ tail s for any non-empty s.

9.2 A model for sequences

I. Concept

  • A sequence is a finite function from ℕ to X whose domain is an initial segment 1..n.
  • Formal: seq X == { s : ℕ ⇸ X | ∃ n : ℕ • dom s = 1..n }.
  • Indexing: s n denotes the nth element — functional application.
  • Squash: compacts a sparse function into a contiguous sequence.
  • Restriction: s ↾ A = squash (s ▷ A) — range-restrict then compact.
  • Head / tail schemas: head s = s 1; #tail s = #s − 1.
  • seq1 X: non-empty sequences — head and tail are total on this set.
  • iseq X: injective sequences — no duplicates.

9.4 Structural induction

I. Concept

  • Induction principle for ℕ: prove P 0; prove P i ⇒ P(i + 1); conclude ∀ n : ℕ • P n.
  • Base case + inductive step: two branches of an induction proof.
  • Inductive hypothesis: assume P t when proving P(⟨x⟩ ⁀ t).
  • Structural induction for sequences: prove P ⟨⟩; prove x ∈ X ∧ P t ⇒ P(⟨x⟩ ⁀ t); conclude ∀ s : seq X • P s.
  • Equational reasoning: chain of equalities, each justified by a named law.
  • Used to prove: filter is distributive, reverse commutes with filter.

  • Base case: (⟨⟩ ⁀ t) ↾ A = t ↾ A = ⟨⟩ ⁀ (t ↾ A) = (⟨⟩ ↾ A) ⁀ (t ↾ A). ✓
  • Inductive step: split on x ∈ A vs x ∉ A; use filter.2, cat.2, inductive hypothesis.
  • Conclusion: ∀ s, t : seq X; A : ℙ X • (s ⁀ t) ↾ A = (s ↾ A) ⁀ (t ↾ A).

Prove reverse and filter commute: reverse(s ↾ A) = (reverse s) ↾ A.

  • Property: induction on s.
  • Base case: established in Example 9.8 of the textbook.
  • Inductive step: uses reverse.2, filter.2, inductive hypothesis, and filter distributivity.
  • Conclusion: ∀ s : seq X; A : ℙ X • reverse(s ↾ A) = (reverse s) ↾ A.

Prove concatenation is associative: s ⁀ (t ⁀ u) = (s ⁀ t) ⁀ u.

  • Property: induction on s.
  • Base case: ⟨⟩ ⁀ (t ⁀ u) = t ⁀ u = (⟨⟩ ⁀ t) ⁀ u by cat.1. ✓
  • Inductive step: (⟨x⟩ ⁀ r) ⁀ (t ⁀ u) = ⟨x⟩ ⁀ (r ⁀ (t ⁀ u)) → apply hypothesis.
  • Conclusion: ∀ s, t, u • s ⁀ (t ⁀ u) = (s ⁀ t) ⁀ u.
