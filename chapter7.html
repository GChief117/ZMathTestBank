Chapter 7 — Relations

7.1 Binary Relations

I. Concept

  • Binary relation from X to Y → set of ordered pairs, subset of X × Y.
  • Type: X ↔ Y == ℙ(X × Y) — all such subsets.
  • Maplet notation: x ↦ y replaces (x, y) in Z.
  • Infix form: _R_ lets you write x R y between arguments.
  • Relations are many-to-many: one x may map to many y, and vice versa.
  • Graphs (arrow diagrams) visualize which pairs exist in a relation.

II. Vocab Table

MapletZ notation x ↦ y for the ordered pair (x, y).
Infix relationRelation written between its arguments using underscores: x R y.
GraphArrow diagram showing which pairs are in the relation.

7.2 Domain and Range

I. Concept

  • Domain: dom R → set of all source elements appearing in R.
  • Range: ran R → set of all target elements appearing in R.
  • Domain restriction: A ◁ R → keep pairs with source in A.
  • Range restriction: R ▷ B → keep pairs with target in B.
  • Domain subtraction: A ⩤ R → drop pairs with source in A.
  • Range subtraction: R ⩥ B → drop pairs with target in B.
  • Relational image: R(| A |) = ran(A ◁ R) → targets from A.

II. Vocab Table

TermPlain English
DomainSet of all source elements appearing in R.
RangeSet of all target elements appearing in R.
Domain restrictionA ◁ R — keep pairs whose source is in A.
Range restrictionR ▷ B — keep pairs whose target is in B.
Domain subtractionA ⩤ R — remove pairs whose source is in A.
Range subtractionR ⩥ B — remove pairs whose target is in B.
Relational imageR(| A |) = ran(A ◁ R) — targets reachable from A.

7.3 Relational Inverse

I. Concept

  • Source / target: for R : X ↔ Y, X is source, Y is target.
  • Inverse: R∼ swaps each pair → y ↦ x for every x ↦ y ∈ R.
  • Homogeneous: source = target type (X ↔ X); otherwise heterogeneous.
  • Identity: id X == { x : X • x ↦ x } — each element to itself.
  • Reflexive: id X ⊆ R — every element related to itself.
  • Symmetric: x ↦ y ∈ R ⇒ y ↦ x ∈ R; implies S = S∼.
  • Antisymmetric: x ↦ y and y ↦ x both in R ⇒ x = y.
  • Asymmetric: x ↦ y ∈ R ⇒ ¬(y ↦ x ∈ R).

7.4 Relational Composition

I. Concept

  • Composition: R ; S chains R then S through intermediate y.
  • Formula: x ↦ z ∈ R ; S ⇔ ∃ y • x ↦ y ∈ R ∧ y ↦ z ∈ S.
  • Transitive: x ↦ y and y ↦ z in R ⇒ x ↦ z ∈ R.
  • Equivalence: reflexive ∩ symmetric ∩ transitive.
  • Equivalence class of a: { x : X | x ↦ a ∈ E } — all elements related to a.
  • Partition: equivalence divides X into disjoint classes.
